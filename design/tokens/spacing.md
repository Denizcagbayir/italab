## Spacing tokens — ITAlab

Scale (prefer these values in CSS/layout):

| Token | px |
|-------|-----|
| `spacing-1` | 4 |
| `spacing-2` | 8 |
| `spacing-3` | 12 |
| `spacing-4` | 16 |
| `spacing-5` | 24 |
| `spacing-6` | 32 |
| `spacing-7` | 40 |
| `spacing-8` | 56 |

**Layout defaults:**
- Page gutter mobile: 16px; tablet+: 18–24px.
- Content max width: ~920px (`.shell`).
- Bottom nav clearance: reserve ~96px padding under main content (`safe-area` aware).
- Touch targets: min 44×44px.

**Usage:** Tight gaps inside controls (1–3); between sections (5–7); page boundaries (4–6 on mobile).
